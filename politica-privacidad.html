Política de privacidad

Aquí encontrarás la política de privacidad tanto de este site oficial, como del software que ponemos a tu disposición.

Política de privacidad del site oficial de BASpeed

En virtud del real decreto 13/2012, José Ignacio Legido Barrios, creador de este sitio web informa al usuario del mismo que este sitio web tiene cookies que son necesarias para el correcto funcionamiento del mismo. Se detallan las cookies usadas en este sitio:

Cookie del sitio web, identificada con el nombre de baspeed, la cual se guarda como aceptación de la condición de que este sitio web pueda usar cookies para su correcto visionado y funcionamiento en el PC del usuario.

Cookies de disqus.com, usadas por el sitio disqus.com y necesarias para poder ofrecer un sistema de comentarios a los usuarios en las entradas del blog de este sitio web. Son identificadas como subdominios de disqus.com y algunos otros dominios distintos usados por disqus.com.

Cookies de Google.com, usadas en concreto en Google Analitycs, para poder ver estadísticas de visitas a la página web (como número de visitantes, países de origen de las visitas, etc).

Si desea eliminar cualquier cookie guardada por este sitio web, puede hacerlo perfectamente siguiendo las instrucciones que se dan en la siguiente página web.

Política de privacidad del software creado por djnacho

El software que se pone a disposición del usuario en esta página web, necesita ser instalado en el dispositivo del usuario (ya sea este un ordenador, o bien un dispositivo móvil Android). Se garantiza al usuario que una vez instalado el software, este no accederá a datos personales ni información privada del usuario contenida en dichos dispositivos.

El software puede acceder a información como la IP del dispositivo del usuario, necesaria para las operaciones de test de velocidad y demás operaciones de diagnóstico de la conexión a internet del usuario. Esta información no se transfiere a ningún sitio, si no que es mostrada en pantalla como información de los tests pertinentes al usuario.
