Características

Aquí tenéis un resumen de las características más importantes de BASpeed de cara al uso por parte de los usuarios.

Última tecnología

BASpeed está creado con Embarcadero­ Delphi. Utiliza componentes de altísima calidad de terceras compañías para obtener un nivel de detalle excelente en el entorno gráfico.

Código fuente abierto

El test de velocidad está basado en una versión del mismo que es de código abierto, accesible por todo el mundo a través de github, para que cualquiera pueda modificarlo y hacerlo más potente si quiere.

La red a tope

En diferentes módulos de la suite BASpeed, se utiliza la tecnología multihilo que permite realizar varias tareas al mismo tiempo, de forma que el programa no se congela si una operación lleva un tiempo prolongado.

Siempre actualizado

El programa BASpeed tiene un actualizador propio que permite que cuando se lanza una actualización, el usuario pueda instalarla sin necesidad de desinstalar la versión anterior.
